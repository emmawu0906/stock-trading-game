## 1. 产品概述

### 1.1 产品定位

**「股民闯关：德明利实战」** 是一款以**闯关模式**为核心的股票交易教育Web应用。产品以A股上市公司**德明利 (001309)** 的真实股价区间为基础，通过**5个递进关卡**，帮助零基础用户从"看懂K线"到"建立完整交易系统"，实现系统性的股票交易学习。

### 1.2 核心理念

| 理念 | 说明 |
|------|------|
| 🎮 **游戏化学习** | 闯关模式降低学习门槛，让枯燥的金融知识变得有趣 |
| 📊 **真实模拟** | 模拟真实A股交易规则（T+1、佣金、印花税、100股/手） |
| 🎯 **渐进式教学** | 5关递进，每关解锁新技术指标，从基础到高阶 |
| 💡 **即时反馈** | 每笔交易实时评分（S-D级），帮助用户理解好坏 |
| 🔒 **零风险** | 虚拟资金交易，完全无真实财务风险 |

### 1.3 一句话描述

> **用德明利(001309)的真实K线数据，通过5关闯关式教学，让股市小白在30分钟内掌握K线、成交量、均线、支撑压力和MACD等核心交易技巧。**

## 2. 市场分析

### 2.1 行业背景

- **中国股票市场投资者数量**: 截至2025年底，A股开户投资者超过 **2.2亿**
- **新增投资者**: 每年新增开户人数约 **1500-2000万**
- **散户占比**: A股市场散户资金占比约 **30%**，但交易量贡献超过 **80%**
- **投资者教育需求**: 中国证监会持续推动投资者教育工作，市场需求巨大

### 2.2 痛点分析

| 痛点 | 描述 | 影响群体 |
|------|------|----------|
| 📖 **知识门槛高** | 传统教材晦涩难懂，专业术语多 | 所有新股民 |
| 💸 **学费高昂** | 实盘试错成本极高，很多人亏损数万才学到教训 | 所有股民 |
| 😴 **学习动力不足** | 看书/看视频容易走神，缺乏互动性 | 年轻人群 |
| 🤷 **学了不会用** | 理论和实战脱节，看了课程但不知道怎么下手 | 大部分学员 |
| ⏰ **时间碎片化** | 忙碌上班族没有大块时间系统学习 | 职场人士 |

### 2.3 市场规模估算

```
目标市场 (TAM): 2.2亿股票投资者
可服务市场 (SAM): 每年2000万新增投资者 + 存量有学习需求的投资者 ≈ 5000万
可获取市场 (SOM): 第一年目标用户 100万，付费用户 10万
```

## 3. 目标用户

### 3.1 核心用户画像

#### 用户A：股市新手小白（占比60%）
- **年龄**: 22-35岁
- **特征**: 刚开户或准备开户，对股票充满好奇但一窍不通
- **需求**: 从零学习基础知识，不想在实盘亏钱
- **场景**: 下班后或周末在手机/电脑上学习30分钟

#### 用户B：有一定经验的散户（占比30%）
- **年龄**: 28-45岁
- **特征**: 有交易经验但缺乏系统知识，经常凭感觉操作
- **需求**: 系统化学习技术分析、完善交易体系
- **场景**: 交易日收盘后用游戏复盘和训练

#### 用户C：金融教育从业者（占比10%）
- **年龄**: 25-40岁
- **特征**: 券商投顾、投资培训机构老师
- **需求**: 辅助教学工具，让学员动手实践
- **场景**: 线下课堂或线上直播中作为互动教学工具

### 3.2 用户旅程

```
发现产品 → 进入首页看到5个关卡 → 点击第1关
→ 选择初始资金（10万-100万）
→ 阅读教学幻灯片（2页）
→ 完成测验（2道选择题）
→ 进入模拟交易界面
→ 根据K线信号决定买卖
→ 获得每笔交易评分和反馈
→ 完成关卡，查看报告
→ 解锁下一关
→ 循环直到通关全部5关
```

## 6. 交易系统设计

### 6.1 K线数据生成

- **数据来源**: 基于德明利(001309)真实价格区间 ¥191-308
- **历史K线**: 每关生成30-50天历史数据，让用户有足够多的K线参考
- **交易日数据**: 根据每关的 phases 配置，生成符合教学目标的K线走势
- **波动模型**: 四层因子驱动（趋势因子 + 波动率因子 + 成交量因子 + 事件因子）

### 6.2 价格生成算法

```
每日K线生成流程:
1. 基础趋势变化 = ±strength × random
2. 波动率叠加 = 基础 × (1 + volatilityFactor)
3. 开盘价 = 前收盘 × (1 + gap)
4. 最高/最低 = 开盘 ± 随机振幅
5. 收盘 = 开盘 × (1 + 日涨跌)
6. 成交量 = 基准量 × volumeProfile × random
7. 价格约束在 191-308 区间内
```

### 6.3 技术指标计算

| 指标 | 算法 | 参数 |
|------|------|------|
| SMA | 简单移动平均 | 5/10/20/30日 |
| EMA | 指数移动平均 | 12/26日 |
| MACD | EMA(12) - EMA(26) | DIF/DEA(9)/柱状 |
| KDJ | 随机指标 | 9日周期 |
| 成交量MA | 成交量简单移动平均 | 5/10日 |

### B. 术语表

| 术语 | 解释 |
|------|------|
| K线 | 显示一段时间内股价开盘/最高/最低/收盘的图表 |
| 阳线 | 收盘价 > 开盘价的K线（红色） |
| 阴线 | 收盘价 < 开盘价的K线（绿色） |
| MA | Moving Average，移动平均线 |
| MACD | 指数平滑移动平均线的差离值 |
| 金叉 | 短期均线上穿长期均线，看涨信号 |
| 死叉 | 短期均线下穿长期均线，看跌信号 |
| 放量 | 成交量显著增大（通常超过前一日1.5倍以上） |
| 缩量 | 成交量显著减小（通常低于前一日0.7倍） |
| 支撑位 | 股价下跌时难以跌破的价位 |
| 压力位 | 股价上涨时难以突破的价位 |
| 止损 | 在亏损达到预设比例时果断卖出以控制损失 |
