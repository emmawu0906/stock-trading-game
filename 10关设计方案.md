## 第1关：认识市场 — 学会"看" 👀

### 整体设计
- **模式**：纯观察模式，**不允许交易**
- **目标**：学会读懂K线和成交量，建立对市场的基本感觉
- **核心体验**：用户每天看一根新K线，预测"明天涨还是跌"

### 教学内容（4页幻灯片）
1. **股票是什么** — 买股票=成为公司小老板，德明利做存储芯片
2. **K线基础** — 阳线（红）/阴线（绿）/十字星，实体和影线
3. **如何读一根K线** — 开高低收是什么意思，大阳线小阴线长上影的含义
4. **价格波动的本质** — 股价是买卖双方博弈的结果，涨=买方强于卖方

### 测验（2题）
1. "开盘250，收盘255，这根K线是什么颜色？" → 红色阳线
2. "连续3根阳线，说明什么？" → 短期内买方力量持续占优

### 游戏玩法
```
1. 系统自动展示一根K线（含OHLCV数据）
2. 界面显示当天K线解读：
   "今天是大阳线(+2.8%)，实体饱满无上影，买方控制全场"
3. 用户选择预测：
   [📈 明天会涨]  [📉 明天会跌]  [📊 不确定/震荡]
4. 点"下一天"揭晓答案
5. 系统给出解释：
   "✅ 你猜对了！连续放量上涨后，动能仍在，第二天大概率继续涨"
   或
   "❌ 你猜错了。虽然今天大涨，但上影线很长+成交量萎缩，
    说明高位有抛压，获利盘开始卖出"
```

### 通关条件
- 10天预测，至少答对6天（60%正确率）
- 不要求100%正确，让用户理解"市场不可完全预测"

### 行情设计
```
Day1-2: 温和上涨（+1.5%, +2%）→ 让用户感受上涨趋势
Day3:   小幅回调（-0.8%）→ 教"涨多了要调整"
Day4-5: 继续上涨（+2.5%, +1.2%）→ 回调后继续涨
Day6:   大幅跳水（-3.5%）→ 教"市场有风险，随时可能跌"
Day7:   反弹（+1.8%）→ 教"大跌后可能反弹"
Day8:   震荡（-0.3%）→ 教"有时候方向不明"
Day9:   放量上涨（+3%）→ 教"放量是重要信号"
Day10:  收长上影（+0.2%）→ 教"上影线=上方有压力"
```

### UI变化
- 隐藏交易面板（买入/卖出按钮）
- 显示"预测面板"替代：3个预测按钮 + 预测结果反馈区
- 左侧显示今日K线解读（自动生成）
- 底部显示预测正确率进度条

## 第2关：第一笔交易 — 学会"做" 🎓

### 整体设计
- **模式**：教练陪伴模式，每一步都有教练提示
- **目标**：完成3次"买入→持有→卖出"的完整流程
- **核心体验**：教练实时告诉用户该做什么，手把手带

### 教学内容（3页）
1. **成交量是市场的"心跳"** — 量大=关注度高，量增价涨=健康上涨
2. **量价关系4种模式** — 量增价涨/量增价跌/量缩价涨/量缩价跌
3. **怎么做一笔交易** — 买入(100股起)/卖出/手续费/T+1概念

### 测验（2题）
1. "放量上涨3%说明什么？" → 资金积极买入
2. "你买了100股每股250元，卖到260元，赚了多少？" → (260-250)×100 - 手续费 ≈ ¥980

### 教练系统设计
```
教练在关键时刻弹出提示卡片：

【第1次交易引导】
Day1: 教练："📊 注意！今天放量上涨2.5%，量价配合良好。
       这是一个不错的买入信号。点击【买入】，先买100股试试水。"
       → 用户买入100股
Day2: 教练："👍 你的股票涨了！现在持仓盈利中。
       别着急卖，好的交易需要耐心。继续观察。"
Day3: 教练："⚠️ 今天出现长上影线，上方有抛压。
       建议卖出获利了结。点击【卖出】。"
       → 用户卖出 → "🎉 恭喜！你完成了第一笔完整交易！赚了¥xxx"

【第2次交易引导】
Day4-5: 教练引导在回调后再次买入 → 持有 → 卖出
        这次给出建议但让用户自己选择时机

【第3次交易引导】
Day6-8: 教练只给出信号提示，让用户自己判断
        "今天的信号你自己分析一下？我这次只给提示不给答案"
```

### 通关条件
- 完成3次完整的买入→卖出操作
- 不要求盈利（第1次交易即使亏钱也算通过，重点是学流程）

### 行情设计（相对友好，确保能完成3次交易）
```
Day1-2: 放量上涨 → 给用户明确的买入机会
Day3:   出现卖出信号 → 教用户止盈
Day4:   小幅回调 → 等待
Day5-6: 再次上涨 → 第二次买入机会
Day7:   震荡 → 教用户持有
Day8:   出现卖出信号 → 完成第二次
后续:   再给一次机会完成第三次
```

### 解锁功能
- 成交量柱状图（副图）

## 第3关：仓位管理 — 学会"控制" ⚖️

### 整体设计
- **模式**：自由交易，但系统对仓位有限制和评分
- **目标**：学会分批建仓，不要把所有钱一次全买进去
- **核心规则**：**单次买入不超过总资金的30%**

### 教学内容（3页）
1. **为什么不能All-in** — 满仓=没有子弹了，跌了无法补仓，涨了无法加仓
2. **仓位三分法** — 底仓(30%) + 追仓(30%) + 机动仓(40%)
3. **分批建仓实战** — 信号一般→试仓10%，信号确认→加仓20%，强信号→最多到60%

### 测验（2题）
1. "你有10万资金，看到一个买入信号，应该买多少？" → 先买1-3万(试仓)
2. "为什么不能满仓操作？" → 留有余地应对万一下跌，可以补仓降低成本

### 系统机制
```
- 仓位监控面板：实时显示当前仓位百分比（用进度条可视化）
  [||||||||          ] 30% 已用仓位
  
- 仓位警告：
  买入超过30%单笔 → 弹窗："⚠️ 你一次买入了35%仓位！
  建议单次不超过30%。确定要这么做吗？"
  
- 仓位评分维度：
  - 是否分批建仓（不是一次全买）→ +20分
  - 单笔最大仓位≤30% → +30分
  - 有加仓行为（在确认信号后加仓）→ +20分
  - 有减仓行为（在风险信号出现时减仓）→ +20分
  - 空仓等待过（不是每天都持仓）→ +10分
```

### 通关条件
- 盈利（任意盈利即可）+ 仓位管理评分 ≥ 70分

### 行情设计
```
Day1-2: 温和上涨（试仓机会）
Day3:   回调（如果满仓就亏了，如果试仓可以补仓）
Day4-5: 明确上涨（加仓机会）
Day6:   大涨后小跌（减仓信号）
Day7-8: 震荡（观望）
Day9-10: 方向选择（让用户用仓位管理的技巧应对）
```

### 解锁功能
- MA5均线、MA10均线

## 第4关：止损纪律 — 学会"认输" 🛡️

### 整体设计
- **模式**：自由交易，但**买入时必须设置止损价**
- **目标**：培养止损习惯，理解"小亏大赚"的核心逻辑
- **核心规则**：每笔买入必须设置止损线（默认-5%，可自定义-3%到-10%）

### 教学内容（3页）
1. **为什么止损是最重要的技能** — 10次交易亏6次也能赚钱，关键是亏小赚大
2. **止损位怎么设** — 技术止损（跌破均线）、百分比止损（-5%）、资金止损（最多亏¥5000）
3. **止损的心理挑战** — "再等等也许会涨回来"是最大的敌人

### 测验（2题）
1. "你250元买入，止损设在237.5元（-5%），这意味着什么？" → 如果跌到237.5系统自动卖出，最多亏5%
2. "止损被触发后股价又涨回来了，说明止损设错了吗？" → 不是，止损是纪律，个别亏损是正常的代价

### 系统机制
```
- 买入时强制弹出止损设置：
  ┌──────────────────────────┐
  │ 🛡️ 设置止损价            │
  │                          │
  │ 买入价: ¥250.00          │
  │ 止损价: [¥237.50] (-5%)  │
  │                          │
  │ 预设: [-3%] [-5%] [-8%]  │
  │                          │
  │ 💡 如果股价跌到止损价，    │
  │    系统将自动卖出保护你    │
  │                          │
  │ [确认买入并设置止损]       │
  └──────────────────────────┘

- 止损触发时：
  "🛡️ 止损触发！你在¥237.50自动卖出了。
   虽然这笔亏了5%，但你保护了95%的本金。
   这就是纪律的力量！"

- 特殊场景（止损后反弹）：
  "💡 注意：止损后股价涨回来了。你可能觉得不该止损。
   但记住：10次止损中有3次能救你避开大跌(>15%)，
   这3次节省的钱远超7次白止损的损失。
   纪律 > 个案。"
```

### 通关条件
- 每笔买入都设置了止损
- 止损被触发时果断执行（不取消）
- 完成至少5笔交易

### 行情设计（特意制造止损场景）
```
Day1-3: 上涨让用户买入
Day4-5: 突然暴跌（触发止损）→ 测试纪律
Day5-6: 止损后继续跌（证明止损是对的）
Day7-8: 反弹上涨，新的买入机会
Day9-10: 小幅回调后继续涨（止损不被触发=持有正确）
Day11-12: 温和下跌到止损附近（再次测试）
```

### 解锁功能
- MA20均线

## 第5关：择时等待 — 学会"忍" ⏳

### 整体设计
- **模式**：自由交易，但评估你是否在好时机出手
- **目标**：学会等待高胜率机会，不冲动交易
- **核心理念**：大部分时间应该空仓等待，只在好机会出现时出手

### 教学内容（3页）
1. **等待是最被低估的能力** — 巴菲特："投资的关键是等待好球来了才出手"
2. **什么是高胜率信号** — 三信号共振：量价配合 + 均线趋势 + 技术形态
3. **信号强度打分** — 1信号=观望，2信号=可试仓，3信号共振=可加仓

### 测验（2题）
1. "连续5天震荡无方向，你应该？" → 空仓等待，方向不明不出手
2. "放量突破+均线多头+MACD金叉同时出现，说明什么？" → 三信号共振，强买入机会

### 系统机制
```
信号强度面板（每天自动更新）：
  ┌───────────────────────────────────────┐
  │ 📡 今日信号分析                        │
  │                                       │
  │ 量价关系: 放量上涨 ✅                   │
  │ 均线趋势: MA5>MA10 ✅, <MA20 ❌        │
  │ 形态信号: 无明显信号 ➖                  │
  │                                       │
  │ 信号强度: ★★☆☆☆ (2/5)               │
  │ 建议: 可以小仓位试探，不宜重仓          │
  └───────────────────────────────────────┘
```

### 通关条件
- 交易胜率（赚钱笔数/总笔数）≥ 50%
- 盈亏比（平均盈利/平均亏损）≥ 1.5
- 这两个指标奖励"等待好机会再出手"的行为

### 行情设计
```
Day1-3: 震荡无方向（考验耐心，别冲动交易）
Day4:   突然放量上涨（真机会来了）
Day5-6: 继续上涨（持有）
Day7:   冲高回落（止盈信号）
Day8-10: 再次震荡（又要等）
Day11:  MACD金叉+放量（第二次好机会）
Day12:  继续涨（持有）
```

### 解锁功能
- 成交量均线（量MA5/MA10）

## 第6关：市场判势 — 学会"顺势" 🌊

### 整体设计
- **模式**：自由交易，行情包含牛、熊、震三种明显阶段
- **目标**：识别当前市场状态，在不同环境采取不同策略
- **核心规则**：牛市重仓，熊市空仓，震荡轻仓

### 教学内容（3页）
1. **三种市场状态** — 上涨趋势/下跌趋势/震荡整理的辨别方法
2. **顺势而为** — 不要在下跌趋势中抄底，不要在上涨趋势中做空
3. **仓位匹配趋势** — 牛市：60-80%仓位，熊市：0-20%仓位，震荡：30-50%仓位

### 测验（2题）
1. "MA5>MA10>MA20且股价在所有均线之上，这是什么状态？" → 多头排列/上涨趋势
2. "下跌趋势中股价突然涨了一天，应该怎么做？" → 不追涨，可能是反弹陷阱

### 系统机制
```
趋势判断面板：
  ┌─────────────────────────────┐
  │ 🌊 当前市场状态              │
  │                             │
  │  🟢 上涨趋势                │
  │  判断依据:                   │
  │  · MA5 > MA10 > MA20 ✅     │
  │  · 股价在MA10上方 ✅         │
  │  · 近5日3涨2跌 ✅           │
  │                             │
  │  建议仓位: 60-80%           │
  │  策略: 逢低买入，持股为主    │
  └─────────────────────────────┘
```

### 趋势评分机制
```
根据用户在不同趋势阶段的仓位：
- 上涨阶段持仓≥50% → +满分
- 上涨阶段空仓 → 扣分（错过机会）
- 下跌阶段空仓或轻仓 → +满分
- 下跌阶段重仓 → 扣分（没有避险）
- 震荡阶段仓位适中(30-50%) → +满分
```

### 通关条件
- 趋势匹配评分 ≥ 70分
- 总体盈利（任意盈利）

### 行情设计（三段式）
```
第一段 Day1-5: 上涨趋势（均线多头排列，应该持仓）
第二段 Day6-10: 转为下跌（均线空头排列，应该空仓）
第三段 Day11-15: 震荡见底后再涨（应该识别底部信号重新入场）
```

### 解锁功能
- MA30均线

## 第7关：对抗贪婪 — 学会"止盈" 💰

### 整体设计
- **模式**：自由交易 + 情境选择
- **目标**：学会在赚钱的时候保住利润，不因贪婪回吐
- **核心技巧**：移动止盈、分批止盈

### 教学内容（3页）
1. **贪婪是怎么让人亏钱的** — "再涨一点再卖"→最后利润全吐回去
2. **止盈策略1：移动止盈** — 赚10%后，止损线上移到成本价（保本）
3. **止盈策略3：分批止盈** — 涨10%卖1/3，涨15%再卖1/3，留1/3追趋势

### 测验（2题）
1. "你买入后已经赚了12%，应该怎么做？" → 至少卖出1/3锁定利润，其余设移动止盈
2. "移动止盈是什么意思？" → 随着股价上涨，不断提高止损价位

### 系统机制
```
- 买入时可以同时设置止盈目标价
- 盈利超过10%时弹出止盈提醒："你已经赚了12%！
  建议: [卖出1/3] [设移动止盈] [继续持有]"
- 贪婪指数监控：
  如果用户拿着高浮盈不卖，且出现顶部信号 → 
  "⚠️ 贪婪预警！你的浮盈已达15%但出现放量滞涨，
   历史上这种情况60%会回调。考虑至少锁定部分利润。"
```

### 行情设计（先甜后苦）
```
Day1-3:  温和建仓机会
Day4-7:  持续上涨（用户赚到10-15%利润）
Day8:    高位放量滞涨（止盈信号！走不走？）
Day9:    小幅回调（给没走的人第二次机会）
Day10:   暴跌回吐大部分利润（不卖的惩罚）
Day11-12: 反弹但不再创新高（教训时刻）
```

### 通关条件
- 总盈利 ≥ 8%
- 止盈评分 ≥ 70分（是否在高位做出了止盈操作）

### 解锁功能
- MACD指标（DIF/DEA/柱状图）

## 第8关：对抗恐惧 — 学会"逆势思考" 🧠

### 整体设计
- **模式**：自由交易 + 恐慌情境
- **目标**：在恐慌中保持理性，不盲目割肉，识别"恐慌底"
- **核心理念**："别人贪婪时恐惧，别人恐惧时贪婪"

### 教学内容（3页）
1. **恐惧如何让人亏钱** — 恐慌割肉→卖在最低点→反弹时不敢买
2. **区分恐慌下跌和趋势下跌** — 缩量下跌=温水煮青蛙，放量暴跌后缩量=恐慌底
3. **逆向思维** — 当别人都在卖的时候，可能就是机会

### 测验（2题）
1. "连跌3天-10%，成交量创10日新高，你应该？" → 观察后续是否出现缩量企稳信号再考虑买入
2. "恐慌底的特征是什么？" → 放量暴跌后成交量萎缩+出现长下影线

### 系统机制
```
恐慌情境弹窗（在暴跌时触发）：
  ┌─────────────────────────────────┐
  │ 😱 市场恐慌！                    │
  │                                 │
  │ 今日暴跌 -4.2%，你的持仓         │
  │ 已亏损 -6.8%                    │
  │                                 │
  │ 你的感受：                       │
  │ "要不要赶紧卖掉？万一继续跌……"  │
  │                                 │
  │ 🧠 冷静分析：                    │
  │ · 成交量放到了3倍→恐慌性抛售     │
  │ · RSI已到超卖区域                │
  │ · 下影线很长→下方有买盘支撑       │
  │                                 │
  │ 你的选择：                       │
  │ [恐慌割肉] [持有观望] [逆势加仓]  │
  └─────────────────────────────────┘
```

### 行情设计
```
Day1-3:  正常上涨，建仓
Day4-5:  突然连续暴跌（-3%, -4%）→ 恐慌测试
Day6:    接着暴跌(-5%) → 最大恐慌点（但这是底部！）
Day7-8:  V形反弹大涨 (+4%, +3%) → 敢加仓的赚钱
Day9-10: 缓慢上涨 → 恢复
Day11:   再次小幅回调 → 第二次恐惧测试
Day12:   企稳反弹
```

### 通关条件
- 在恐慌情境中做出理性决策（不在最低点割肉）
- 没有"报复性交易"（恐慌后立即反向操作）

### 解锁功能
- KDJ指标

## 第9关：信息解读 — 学会"分析" 📰

### 整体设计
- **模式**：新闻+交易模式，每天有市场新闻/消息推送
- **目标**：学会判断新闻消息对股价的真实影响
- **核心能力**：区分利好利空、判断消息真实影响力

### 教学内容（3页）
1. **消息面分析入门** — 宏观政策/行业政策/公司公告 三个层面
2. **利好不一定涨** — "利好出尽是利空"、"靴子落地"效应
3. **消息解读方法** — 消息的时效性、预期差、量价配合验证

### 测验（2题）
1. "公司发布业绩增长30%的季报，但股价高开低走放量下跌，为什么？" → 利好可能已提前被消化了，资金借利好出货
2. "国家出台芯片行业扶持政策，你应该？" → 关注个股是否真正受益+观察量价反应再决策

### 新闻系统设计
```
每天推送1-2条新闻，用户需要判断影响：

📰 "德明利获得存储芯片国产替代大单，预计增加营收20%"
   → [重大利好📈] [一般利好] [影响不大] [利好出尽📉]

答案揭晓：
   "✅ 这是重大利好！关键看量价验证：
    如果次日放量大涨→说明市场认可，可以买入
    如果放量高开低走→说明资金借利好出货，观望"

典型新闻类型：
1. 公司财报（营收/利润/增速）
2. 行业政策（芯片补贴/出口管制）
3. 大股东动态（增持/减持/质押）
4. 机构评级（券商研报/目标价）
5. 市场传闻（未经证实的消息）
6. 宏观环境（加息/降息/GDP数据）
```

### 通关条件
- 新闻判断正确率 ≥ 60%
- 综合盈利（任意盈利）

### 行情设计
- 行情走势与新闻强关联
- 有时新闻和走势一致（利好→涨），有时不一致（利好→跌）教用户灵活判断

### 解锁功能
- 布林带指标
